# BASHIUM
This repository contains a Python program with graphical user interface (GUI) to manage and run various configuration scripts for Debian-based systems. The program uses 'tkinter' for the GUI

## Prerequisites

Before running the program, you need to install tkinter
sudo apt install python3-tk

The program provides a graphical interface to run configuration scripts for Debian system.
